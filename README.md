# Software Portfolio

Dieses Repository enthält ausgewählte Codebeispiele aus eigenen
Softwareprojekten mit Fokus auf C#, objektorientierte Programmierung
und modulare Systemarchitekturen.

Die enthaltenen Module sind bewusst kuratiert und dienen dazu,
Architekturentscheidungen, saubere Code-Struktur und praktische
Problemlösungen zu demonstrieren.

## Inhalte

- EntitySystem  
  Modulares Entity-Component-System mit klarer Trennung von Core-Logik,
  Serialisierung und Beispielen.

- SettingsSystem  
  Datengetriebenes und erweiterbares Settings-System mit optionalen
  UI-Integrationen als Beispiele.

Die Systeme sind unabhängig voneinander aufgebaut und jeweils
für sich verständlich.
